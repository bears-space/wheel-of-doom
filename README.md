# Wheel of Names (Vue + shadcn-style UI)

A configurable wheel-of-names app built with Vue 3, Vite, Tailwind, and shadcn-inspired components.

## Features

- Editable name list (one entry per line)
- Spin controls:
  - rotation speed (turns/second)
  - spin duration (seconds)
- Winner selection when spin ends
- Import names from:
  - CSV (`.csv`)
  - Excel (`.xls`, `.xlsx`)

## Run

```bash
npm install
npm run dev
```
